# Prompt Quality Checklist

Run this before finalizing a prompt.

## Required Checks

- The subject is specific enough to draw.
- The environment has time, place, and spatial cues.
- The lighting has source, direction, color, and mood.
- The material layer includes tactile surface details.
- The camera/composition layer controls framing.
- The style references are coherent.
- The emotional or era tone is explicit.
- Negative constraints target likely failure modes.
- The language strategy fits the target model.
- The prompt has a clean copy-ready version.

## Diagnostic Questions

- If the image looks flat, is the lighting layer specific?
- If the image looks generic, are style anchors too weak?
- If the image looks messy, are there too many competing concepts?
- If the character looks like a fashion model, did the pose/action instructions become too commercial?
- If a series lacks consistency, did the prompt define color, medium, composition, and narrative systems?
- If the model ignores later details, can the prompt be shortened or moved earlier?

## High-Leverage Iteration Knobs

1. Subject specificity.
2. Light source and contrast.
3. Camera distance and lens feel.
4. Color science and palette.
5. Medium or material cues.
6. Repeated style anchors.
7. Negative constraints.
8. Model-specific length and language.
